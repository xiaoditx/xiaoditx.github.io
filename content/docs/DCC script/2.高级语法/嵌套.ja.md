### JSONネスティング

DCCは一定の方法でJSONをネストすることができるため、シンタックスハイライトはJSON構文をサポートする必要があります。JSONを埋め込む場合、前の行と次の行に対してインデントがあります。具体的なインデント量は厳密に規定されていません。例えば：

```DCC
前语句
    {
        "name": "Tom"
    }
后语句
```

ネストされたJSONコードは、JSON構文を使用してハイライトする必要があります

### C++ネスティング

+++を使用してC++コードをラップすることができます。これらの3つの文字は1行を占有し、コメントは許可されません

```DCC
前语句
+++
#include<iostream>
int main(){
    std::cout<<"hello word";
}
+++
后语局
```

C++コードはファイルの任意の位置で定義することができ、グローバルに呼び出すことができます
