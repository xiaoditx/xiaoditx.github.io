- [x] Display danmaku on desktop
- [ ] Support importing danmaku content from json files
- [ ] Support custom danmaku transparency

### Build Method

Use the following command to compile the DEBUG version of the software

```batch
make
```

Use the following command to compile the DEBUG version of the 32-bit software

```batch
make ARCH=32
```

For other operations, please use `make help` in the project root directory

The makefile was ported from the [keybonk project](https://github.com/xiaoditx/keybonk) and may have unused targets, so it's best not to use them as they are untested
